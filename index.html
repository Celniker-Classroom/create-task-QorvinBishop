Affine Cipher

This cipher uses a 2-character key (a, b) consisting of 2 integers that represent the coefficients in the linear function: f(x) = (ax + b) mod 26. For example, a key of (1, 2) would result in the equation f(x) = (x + 2) mod 26. (Note that letting a = 1 just creates a Caesar cipher.)

We assign the letters A-Z numerical values in increasing order, where A = 0 and Z = 25. The encrypted message is based on the function above (where the original values of the letters are inputs). The outputs are then converted back to letters based on the letter-to-number assignment.

To decode an affine cipher, the value of the coefficient "a" must be coprime to 26. (Note that if non-integers in range are inputted for keys, they will be rounded down to the nearest integer.)
